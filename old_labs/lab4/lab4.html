Preguntas a resolver

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

La unica semejanza es el nombre, con Java se pueden crear aplicaciones que se ejecutan en una maquina o explorador virtual, mientras que JavaScript solo se ejecuta en el explorador

Java es un lenguaje de programacion OOP, mientras que JavaScript es un lenguaje scripts OOP

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Date.now(), getDate(), getMonth(), getFullYear(), getHours(), getMinutes()

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

length, push, pop, shift, unshift

¿Cómo se declara una variable con alcance local dentro de una función?

Con la palabra reservada let

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Que por accidente cambiemos el valor de esa variable mas adelante en el codigo por error humano, ya que podemos modificar el valor de la variable por accidente sin darnos cuenta, y con let nos estamos "obligando" a hacer las cosas bien
